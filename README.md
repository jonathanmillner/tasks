# Tasks app frontend / React, TypeScript, Tailwind CSS, Vite

### Create an .env file
```
VITE_API_URL=[YOUR_BACKEND_URL]

e.g. 
[YOUR_BACKEND_URL] -> `http://localhost:3001`
```

### Run the app
```
yarn run dev
```
